# Events Startup Project

An event booking web application built with React and Vite.
Users can browse tech events, search and filter events, view event details, manage a shopping cart, authenticate, and complete a checkout flow.

## Live Demo

https://events-startup-project-app.onrender.com/

## Features

* Browse and search events
* Filter events by category
* View detailed event pages
* Add tickets to cart
* Update ticket quantities
* Authentication flow (login/logout/register)
* Checkout flow
* Loading and error handling

## Authentication

The project uses React Context API and localStorage for frontend authentication state management.
json-server and json-server-auth were used for mock API and authentication flow practice during development.
